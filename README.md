### 1. 커밋 유형 지정
| 이모티콘 | 문자 | 커밋 유형 | 의미 |
| --- | --- | --- | --- |
|  ✨ | : sparkles : | feat | 새로운 기능 추가 |
| 🐛 | : bug : | fix | 버그 수정 |
| 📝 | : memo : | docs | 문서 수정 (md 파일) |
| ♻️ | : recycle : | refactor | 코드 리팩토링 |

### 2. 커밋 메시지
```
✨ feat: 로그인 개발 (#issue_number)
```

### 3. 브랜치 전략
```
|-- main
    |-- develop ✅ gitaction target
        |-- feature/<# issue_number>
```
